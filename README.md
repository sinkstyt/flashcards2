## Description

_This page was built to tinker with some jQuery tidbits that the Epicodus curriculum introduced. The terms that the project hoped to help its users master include: JavaScript (JS), Operators, Variables, Variable naming conventions, Functions, Methods, Arguments, Parameters, Return, Chaining methods, Strings, Booleans, Undefined, NaN, Null, Alert, Comments, jQuery, Attributes. For inspiration and reference, the creators would like to credit the fine community on Stack Overflow and their fine work in styling and creating predecessor flashcard sites._
